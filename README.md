# 🎮 Рулетка демонов с KotGDPS

Интерактивная веб-игра, вдохновлённая Geometry Dash, где игроку предстоит последовательно проходить уровни возрастающей сложности, накапливая «баллы стойкости» по уникальной системе расчёта. Проект предназначен для приватного сервера **KotGDPS** и доступен на **GitHub Pages**.

## 🔥 Особенности

- Случайный выбор уровней из базы данных (пользовательские + официальные).
- Два режима прогресса: «Без флюков» (линейное увеличение) и «С флюками» (ручной ввод процента).
- Гибкие настройки пула уровней: только демоны, включая уровни ниже Easy Demon и без звёзд.
- Система баллов стойкости с модификаторами за сложность, прогресс и длину уровня.
- Возможность пропустить уровень (с нулевым или отрицательным эффектом).
- Подробная история всех появившихся уровней и финальная статистика.
- Полностью адаптивный дизайн для мобильных устройств.

## 🎯 Как играть

1. На главном экране выберите желаемые настройки:
   - **Типы уровней** – только пользовательские или с добавлением уровней RobTop.
   - **Побочные типы** – разрешить уровни ниже Easy Demon (Insane, 9★) и/или уровни без звёзд (N/A).
   - **Режим прогресса** – без флюков (+1% за каждый пройденный уровень) или с флюками (вы вводите достигнутый процент, следующий порог = введённое + 1).
   - **Негативный пропуск** – при включении пропуск уровня списывает баллы (равные стоимости уровня), иначе пропуск бесплатный.
2. Нажмите **«Начать пытку»**.
3. На игровом экране вы увидите случайно выбранный уровень, его автора, сложность, набор скиллсетов и требуемый процент.
4. Чтобы пройти уровень, нажмите **«Дальше»** (в режиме с флюками предварительно введите достигнутый процент). Если вы не хотите проходить текущий уровень, используйте **«Пропуск»**.
5. При достижении требования 100% начнётся финальный тест – после его прохождения игра завершается победой.
6. В любой момент можно **«Сдаться»** – игра завершится досрочно с отображением прогресса.

## 📊 Правила подсчёта баллов стойкости

| Сложность                | Базовые баллы |
|--------------------------|---------------|
| Easy Demon               | 5             |
| Medium Demon             | 10            |
| Hard Demon               | 15            |
| Insane Demon             | 20            |
| Extreme Demon            | 30            |
| Ниже Easy Demon (Insane) | 2.5           |
| Без звёзд (N/A)          | индивидуально |

**Модификаторы прогресса (для обычных демонов):**
- Прогресс ≤ 10% → всегда +5 баллов.
- Прогресс < 25% → базовые баллы делятся на 2.
- Прогресс > 50% → +5 баллов.
- Длинный уровень + прогресс > 50% → ещё +5 баллов.

**Для уровней ниже Easy Demon:**
- Прогресс < 25% → 0 баллов.
- Прогресс ≥ 25% → фиксированные 2.5 балла.

**Особые случаи:**
- Уровень «Silent penis» (N/A) приравнен к Hard Demon (база 15) и рассчитывается по общим правилам для демонов.

Все расчёты отображаются в строке «Баллы стойкости» с подробным пояснением.
